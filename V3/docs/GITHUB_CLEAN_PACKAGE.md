# GitHub Clean Package Notes

This package is designed to be small enough and clean enough for GitHub while
remaining reproducible from the included strict experiment data and scripts.

## Included

- Experiment code.
- Strict ready data.
- Audit reports.
- Prepared JSONL files.
- Translation CSV files.
- Evaluation metrics.
- Paper tables.
- Run configuration files.

## Excluded

- LoRA adapter weights.
- Training checkpoints.
- Optimizer, scheduler, scaler, RNG, and trainer states.
- Tokenizer/model cache files.
- Original PDF source.
- Python caches and temporary files.

The excluded model artifacts can be regenerated by rerunning the experiment
scripts in the `YYQEI` conda environment.

## Why Model Artifacts Are Excluded

Model artifacts are large, unnecessary for source-code review, and may carry
additional redistribution constraints from third-party model licenses. Keeping
them out of GitHub makes the repository easier to clone and safer to publish.

Users who need trained adapters should publish them separately with explicit
model-card, license, and data-use documentation.
